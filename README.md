# diaxyz

**diaxyz** is a retailer of industrial storage racking systems
(https://en.wikipedia.org/wiki/Pallet_racking) ranging from low-tech to complex
high-tech fully automated/robotic custom solutions.

We also offer consulting, installation and inspection services for any storage
systems.

## Installation

Clone the repository

```bash
git clone https://github.com/diaxyzdev/diaxyz.git
cd diaxyz
./configure
```

## Project Structure

```
- cloud/ (directory synced with cloud storage provider)
- infrastructure/ (infrastructure as code)

- forstorebg.com (The Bulgarian version of the companies public facing website)
- .simba (library of usefull reusable functions for bash scripts)
- configure (script for configuring this repository)
- Makefile (configuration stage artifact)
```
